# DC-LLM-Base

本仓库用于记录数据中心基础研究小组在实训期间的研究成果与代码实现，重点关注大语言模型（LLM）与自然语言处理（NLP）的基础技术探索。

---

## 📌 项目简介
本项目主要研究数据中心环境下 LLM 的基础应用与技术链路。内容涵盖：
* **数据预处理**：针对大模型训练语料的清洗、分词及去重技术。
* **基础架构调研**：探讨 NLP 模型在数据中心部署的底层逻辑。
* **实验复现**：对常用 NLP 基准模型进行测试与评估。

## 📂 目录结构
* `/src`: 核心研究代码与脚本。
* `/docs`: 技术调研文档与实训心得。
* `/configs`: 实验参数配置文件。
* `requirements.txt`: 项目依赖库列表。
